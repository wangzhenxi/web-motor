
# web-motor

开发者效率工具，解决一些开发中的痛点

## 开发

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build
```

## 功能描述

### 图片压缩

- 通过遍历文件树，读取文件后缀，根据筛选的图片格式进行压缩
- 会直接将压缩文件替换掉源文件
- 压缩方案源自[手搓一个TinyPng压缩图片](https://juejin.cn/post/6882551009219575815)
